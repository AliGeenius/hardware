##### GPIO
* GPIO1_21  ->  Step X

* GPIO2_4   ->  Dir Y
* GPIO1_23  ->  Step Z
* GPIO2_3   ->  Dir Z

* GPIO2_2   ->  Dir Ext1
* GPIO1_24  ->  Step Ext2
* GPIO2_5   ->  Dir Ext2
* GPIO1_25  ->  Step Ext3
* GPIO3_7   ->  Dir Ext3
* GPIO1_27  ->  Step User
* GPIO3_8   ->  Dir User
